Welcome to my interactive 3D portfolio! I'm **Govind Dangi**, a Java Full Stack Developer and Software Engineer currently pursuing my B.Tech in Computer Science and Engineering (2023–2027). I specialize in building scalable web applications and backend systems, with a strong foundation in Java, Spring Boot, React, and MySQL.

This repository contains the source code for my personal portfolio website, built with a focus on modern design, 3D interactivity, and smooth animations to provide a premium user experience.

## ✨ Key Features

- **Interactive 3D Elements:** Powered by Three.js and React Three Fiber for a fully immersive, drag-to-explore experience.
- **Modern Animations:** Fluid transitions, scroll effects, and micro-interactions using Framer Motion and Lenis.
- **Custom Design:** A tailored, premium user interface featuring dynamic glassmorphism, animated backgrounds, and a cohesive dark theme.
- **Responsive Layout:** Carefully optimized to look and perform great on both desktop and mobile devices.
- **Terminal Simulator:** Built-in interactive terminal experience for developers.

## 🚀 Getting Started

To run this project locally, follow these steps:

### Prerequisites
- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/govinddangi4564/My-Portfolio.git
   cd My-Portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173/` (or the port specified by Vite).

## 📂 Project Structure

- `src/components/` - React components for various sections (Hero, About, Projects, etc.)
- `src/data/` - Static data files representing the portfolio data (projects, skills, etc.)
- `src/hooks/` - Custom React hooks for media queries and interactions
- `public/` - Static assets like images, 3D models, and resumes
